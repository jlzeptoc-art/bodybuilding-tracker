How to Use This Tracker
  • Navigate to any week using the tabs above. Select the workout day to view that session's exercises.
  • For each set, enter the weight you used and the reps completed. Data saves automatically to your browser.
  • Exercise names are clickable links (where provided) so you can watch demo videos before each workout.
  • Weeks 1–5 = Foundation Block. Weeks 6–12 = Ramping Block. The program structure changes between blocks.
  • Data is stored locally in your browser. Use the same browser/device to retain your progress across sessions.
⚠ Important Program Notes — Read Before Starting
  • Perform a full general warm-up and exercise-specific warm-up every workout as outlined in the Warm-Up section (should only take 5–10 mins max).
  • For the first week, most sets are taken to an RPE of ~6–7. This means you will be leaving 3–4 reps in the tank on most exercises. This only lasts for the first week (to serve as a deload/intro week). After the first week, the intensity will increase and you'll start taking most sets to ~2–3 reps shy of failure.
  • All working sets are broken up into Early Sets and Last Sets. The Early Sets are all the sets that come before your Last Set. Usually the Early Sets should be a little bit easier than the later sets (slightly lower RPE) whereas the Last Set is pushed closer to failure. In the first two weeks of the program, there will only be one working set for each exercise, so there is only a Last Set RPE.
  • Each exercise name is a clickable link which demonstrates how to perform the exercise. Watch each exercise demo before starting the workout.
  • All other aspects of the program, including when to make an exercise substitution and how to progress through the rep ranges given, is explained in the Bodybuilding Transformation System Guidebook. Give it a full read before starting your first workout!

Exercise-Specific Warm-Up
Perform the following exercise-specific warm-up according to the number of warm-up sets listed in the program.
1 Warm-Up Set Use ~60% of your planned working weight for ~6–10 reps (or until you feel warm and loose)
2 Warm-Up Sets Mini warm-up pyramid:
Set #1 = ~50% of planned working weight for ~6–10 reps
Set #2 = ~70% of planned working weight for 4–6 reps
3 Warm-Up Sets Full warm-up pyramid:
Set #1 = ~45% of planned working weight for ~6–10 reps
Set #2 = ~65% of planned working weight for 4–6 reps
Set #3 = ~85% of planned working weight for 3–4 reps
4 Warm-Up Sets Full warm-up pyramid:
Set #1 = ~45% of planned working weight for ~6–10 reps
Set #2 = ~60% of planned working weight for 4–6 reps
Set #3 = ~75% of planned working weight for 3–5 reps
Set #4 = ~85% of planned working weight for 2–4 reps
